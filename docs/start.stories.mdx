Компонент слайдера

[Посмотреть на GitHub](https://github.com/consta-design-system/slider)

## Быстрый старт

#### Установите пакет и зависимости

`yarn add @consta/uikit @consta/slider` или `npm install @consta/uikit @consta/slider`

#### Подключите тему

Тему обязательно нужно подключать в корне приложения, так, чтобы все компоненты `@consta/slider` были вложены в компонент `Theme`.

[Как подключить тему](http://uikit.consta.design/?path=/docs/thematization-how-to-connect-theme--page)
