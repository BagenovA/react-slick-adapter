```tsx
import { ReactSlickControl } from '@consta/react-slick-adapter/ReactSlickControl';
```

С помощью компонента `ReactSlickControl` можно переопределить дефолтные контролы.

## Свойства

| Параметры | Тип              | По умолчанию | Описание     |
| --------- | ---------------- | ------------ | ------------ |
| `type?`   | `'prev', 'next'` | `prev`       | Тип контрола |

## Пример

```tsx
import React from 'react';
import Slider from 'react-slick';

import { ReactSlickControl } from '@consta/react-slick-adapter/ReactSlickControl';
import { cnReactSlick } from '@consta/react-slick-adapter/ReactSlick';

const items: string[] = [Image1, Image2, Image3, Image4];

export const ReactSlickControlExample = () => {
  return (
    <Slider
      dots
      className={cnReactSlick()}
      nextArrow={<ReactSlickControl type="next" />}
      prevArrow={<ReactSlickControl type="prev" />}
    >
      {items.map((image) => (
        <div>
          <img alt="" src={image} />
        </div>
      ))}
    </Slider>
  );
};
```
