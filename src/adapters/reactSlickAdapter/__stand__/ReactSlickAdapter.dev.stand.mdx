## Создание слайдера

Для того чтобы подключить слайдер необходимо импортировать `reactSlickAdapter` из пакета `@consta/react-slick-adapter`, а также `Slider` из пакета `react-slick`.

```tsx
import React from 'react';
import Slider from 'react-slick';

import { reactSlickAdapter } from '@consta/react-slick-adapter/reactSlickAdapter';

const items: string[] = [
  'https://4kwallpapers.com/images/walls/thumbs_2t/5894.jpg',
  'https://images.unsplash.com/photo-1581610186406-5f6e9f9edbc1?ixlib=rb-4.0.3&ixid=MnwxMjA3fDB8MHxjb2xsZWN0aW9uLXBhZ2V8MnwyMjAzODh8fGVufDB8fHx8&w=1000&q=80',
  'https://4kwallpapers.com/images/wallpapers/macos-big-sur-apple-layers-fluidic-colorful-wwdc-stock-2880x1800-1455.jpg',
  'https://www.gannett-cdn.com/presto/2019/09/24/USAT/ca65ec63-180e-4edd-b92e-38cd0af795a9-catalina.jpg?crop=1279%2C720%2Cx0%2Cy278&width=1200',
];
export const ReactSlickAdapterExample = () => {
  const settings = reactSlickAdapter({ dots: true, arrows: true });

  return (
    <Slider {...settings}>
      {items.map((image) => (
        <div>
          <img alt="" src={image} />
        </div>
      ))}
    </Slider>
  );
};
```

## Индикаторы

Вы можете использовать индикаторы по умолчанию, указать их тип и положение, а также переопределить и указать свои собственные.

### Позиционирование

Позиция индикаторов задаётся через свойство `indicatorsPosition`, которое принимает значения `bottom` и `left`. Значение по умолчанию `bottom`.

```tsx
<Slider {...reactSlickAdapter({ indicatorsPosition: 'bottom' })}>
    {items.map((image) => (
        <div>
            <img alt="" src={image} />
        </div>
    ))}
</Slider>
<Slider {...reactSlickAdapter({ indicatorsPosition: 'left' })}>
    {items.map((image) => (
        <div>
            <img alt="" src={image} />
        </div>
    ))}
</Slider>
```

<ReactSlickAdapterExampleIndicatorsPosition />

### Тип

Есть 4 разновидности индикаторов: `dots`, `number`, `icon`, `photo`. По умолчанию `dots`.

Чтобы выбрать один из них, необходимо определить свойство `indicatorsType`.

> Если вы используете тип `icon` или `image`, укажите свойство `items`, а также, при необходимости, геттеры `getItemIcon` и `getItemImage` для иконок и изображений соответсвенно.

### Свойства адаптера

| Свойство              | Тип и значения                                                  | По умолчанию | Описание                                              |
| --------------------- | --------------------------------------------------------------- | ------------ | ----------------------------------------------------- |
| `items?`              | `ITEM[]`                                                        | -            | Список элементов                                      |
| `getItemImage?`       | `(item: ITEM) => string `<code>&#124;</code>` undefined`        | -            | Результат функции используется в качестве изображения |
| `getItemIcon?`        | `(item: ITEM) => IconComponent `<code>&#124;</code>` undefined` | -            | Результат функции используется в качестве иконки      |
| `indicatorsPosition?` | `'bottom', 'left'`                                              | `'bottom'`   | Позиционирование индикаторов                          |
| `indicatorsType?`     | `'dots', 'number', 'icon', 'image'`                             | `'dots'`     | Тип индикаторов                                       |
